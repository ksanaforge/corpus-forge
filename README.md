# Corpus-Forge

==first time setup==
    cd buildscript
    ksana-bundle
    react-bundle
    cd ..

==start development==
    npm start

==generate release bundle.js==
    npm run build

   double click index.html

==open corpus automatically with hash tag==

  	localhost/corpus-forge/#c=corpus-id

===debugging ksana core==
    cd buildscript
    ksana-bundle-debug.cmd
